LOADER

Loader là chương trình hệ điều hành dùng để nạp (load) chương trình từ ổ cứng vào RAM để CPU có thể chạy. Nó chuẩn bị mọi thứ khi bạn nhấp đôi một file .exe

🔀 So sánh Loader vs Linker
⏰ Khi nào?
Khi bạn chạy chương trình
⚙️ Làm gì?
Nạp từ Disk → RAM
📥 Input
.exe, .dll, .bin
👤 Tạo bởi
Hệ điều hành (Windows, Linux)
⏰ Khi nào?
Khi bạn biên dịch code
⚙️ Làm gì?
Kết hợp .obj → .exe
📥 Input
.obj, .lib, .a
👤 Tạo bởi
Lập trình viên (GCC, MSVC)
🔄 Quy trình từ Code đến Chạy
1️⃣
Viết Code
Bạn viết file main.c, utils.c → Compiler biên dịch
2️⃣
Compiler tạo Object Files
main.c → main.obj, utils.c → utils.obj (Linker sẽ làm việc ở bước tiếp)
3️⃣
Linker kết hợp
main.obj + utils.obj + stdio.lib → program.exe (Linker đảm nhận việc này)
4️⃣
Bạn chạy .exe
Nhấp đôi program.exe → Loader nạp vào RAM → CPU chạy
